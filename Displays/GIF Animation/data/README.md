# GIF Animation Files

Place your GIF files here with the name `animation.gif`.

## Requirements

- **File name:** `animation.gif` (exact name required)
- **Resolution:** 240×240 pixels (recommended for the round display)
- **File size:** Max ~1 MB
- **Format:** Standard GIF (GIF87a or GIF89a)

## How to Use

1. Place your GIF file in this folder
2. Rename it to `animation.gif`
3. In PlatformIO, click **Upload Filesystem Image** (in the PlatformIO sidebar)
4. Done! The animation will play on the display

## Design Tips

- **Round display:** Consider circular or centered artwork that fits the round screen shape
- **Color palette:** GIFs work best with 256 colors or fewer
- **Frame rate:** Keep frame delays between 10–100 ms for smooth playback
- **Test locally:** Preview your GIF at full speed before uploading to the device

## Example Workflow

Replace `animation.gif` with your own → Click **Upload Filesystem Image** → Your animation plays!

No code changes needed.
